2026-07-21

I got rid of the /ɬ/ voicing assimialtion :-) (/ɬ/ -> [ɮ] allophone near other voiced sounds)
I also made it so that the stop-final aspiration only applies to voiceless stops. Aspirated voiced stop is… I don't like it.

That's all! :P Tiny update. I should mention that voicing-concerned allophones can now only devoice voiced consonants, and can no longer voice voiceless consonants (cough cough /ɬ/ assimilation cough cough). I made note of this in Phonology.

The next update will actually matter I promise qwq

2026-07-14: Morphology updates

Second update! And there are some… rather large changes taking place.

The gender system has been completely rebuilt, because I didn't really like it; it felt too “okay so what if [European language] was more complicated?”.
It now uses a tree-like system, as described over in Morphology.

I've also… still not written the conjugation system, despite that being my favourite part of the language. I'll get to it eventually, I just haven't.

Now, this isn't just a status update. I actually almost forgot that while writing this, but it's not just a status update.
The number system is subject to change, because I do plan on adding a dual/plural split, meaning there'd be 5 states of number (since mixed and collective still must be accounted for, so there'd be singular, mixed dual, collective dual, mixed plural, and collective plural)

. That's all! I look forward to documenting, and ultimately working on, the conjugation.
