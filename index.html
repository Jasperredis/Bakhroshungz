[ˈbax.r̥ɔ.ʂuŋz]

Bakhroshungz is a highly agglutinative constructed language with a complex verb paradigm, extensive case system of 31 cases, 8 genders, and a native imperfect abjad, named Bakhrosheqif. It is fictionally spoken by aliens and humans alike. It uses only nouns and verbs; all other things that are to be communicated are done so using various morphologic features.
All pages describe the Central Bakhroshungz dialect (unless noted in Dialects.)
