<h1 align="center">Bakhroshungz</h1>
Bakhroshungz is a highly agglutinative constructed language with a complex verb paradigm, extensive case system of 31 cases, five genders, and a native imperfect abjad, named Bakhrosheqif.  

This repository contains the website for documenting and explaining the grammar, lexicon, phonology, writing system, and world of this conlang.
